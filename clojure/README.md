Clojure
=======

Build
-----

Uses [Leiningen](http://leiningen.org/) as the build tool.

```bash
lein new app clojure-example
cd clojure-example
```

Test
----

Note that lein makes a default test that *fails*.

```bash
lein test
```

Git
---

```bash
git add .
git commit -m "Adds sample files"
```
